Access the repository by reading files directly from GitHub Raw URLs rather than browsing the GitHub HTML pages.

Method:

1. Treat the provided GitHub directory URLs as the root paths for analysis.

2. Enumerate files recursively under each directory. Include:

3. For any file URL, convert the normal GitHub URL into a raw content URL before reading it.

Conversion rule:
Replace:
https://github.com/

With:
https://raw.githubusercontent.com/

Also remove:
/blob/

Example:
Input:
https://github.com/14ag/VirtuaCam/blob/main/software-project/src/VirtuaCam/Main.cpp

Converted raw URL:
https://raw.githubusercontent.com/14ag/VirtuaCam/main/software-project/src/VirtuaCam/Main.cpp

4. Always fetch the raw file contents, not rendered HTML pages. Raw content is required for parsing source code accurately.
